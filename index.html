Online in 5min met Heroku en GitHub?

  • Eenmalig: Maak gratis accounts aan op GitHub.com en Heroku.com
  • Eenmalig: Installeer GitHub Desktop + de commandline tools
  • Eenmalig: Installeer de Heroku Command Line Interface (CLI)
  • We gaan er van uit dat je een Node app lokaal in een folder op je computer hebt staan. Je hebt je terminal venster open en je bent met het cd-commando naar de folder met je app gegaan.
  • Je app bevat een bestand met naam Procfile (met de lijn web: node index.js ) en in je index.js heb je de regels (app.set('port', (process.env.PORT || 5000)); en app.listen(app.get('port'), function() { });).
  • Je zorgt voor een bestand met naam .gitignore met daarin volgende regels:
    /node_modules
    npm-debug.log
    .DS_Store
    /*.env
  • Je bent in de terminal ingelogd bij heroku: heroku login
  • ENKEL DE EERSTE KEER BIJ EEN NIEUWE APP:
  • Stap 1: Maak een nieuwe repository aan via de GitHub website. Je kiest enkel de naam, en laat de voorgestelde settings ongemoeid.
  • Stap 2: In je terminal in je app folder: git init
  • Stap 3: In je terminal in je app folder: git add .
  • Stap 4: In je terminal in je app folder: git commit -am "mijn eerste commit"
  • Stap 5: In je terminal in je app folder: git remote add origin https://github.com/jouwgebruikersnaam/jouwrepositorynaam.git
  • Stap 6: In je terminal in je app folder: git push -u origin master
  • Stap 6: In je terminal in je app folder: heroku create
  • Stap 7: In je terminal in je app folder: git push heroku master
  • EENS DE VORIGE STAPPEN DOORLOPEN ZIJN, BIJ VOLGENDE WIJZIGINGEN:
  • Stap 1: In je terminal in je app folder: git add -A
  • Stap 2: In je terminal in je app folder: git commit -am "boodschap die de aanpassing kort beschrijft"
  • Stap 3: In je terminal in je app folder: git push origin master
  • Stap 4: In je terminal in je app folder: git push heroku master

  • cd (terug naar home folder)
